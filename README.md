## 📌 What is kicad-happy?

kicad-happy is a software tool designed to help you with your electronics design using KiCad. It uses AI to:

- Analyze your schematics for errors or improvements
- Review your PCB layouts for common issues
- Download datasheets for components you use
- Find and suggest sources for components
- Prepare your board files for manufacturing

You don’t need to know how to code or be an expert in electronics. kicad-happy guides you step-by-step.

## 🚀 Running kicad-happy for the First Time

When you start kicad-happy, you will see a simple user interface with options related to KiCad files.

- To analyze a schematic, click **Open Schematic** and find your KiCad `.sch` file.
- For PCB review, click **Open PCB Layout** and select your `.kicad_pcb` file.
- To download datasheets for components, enter the component name or ID in the search box.
- Use the **Component Source Finder** to locate where you can buy parts.
- When ready for fabrication, use the **Fabrication Prep** tool to export your files.

The app provides clear instructions for each step.

## 🛠️ Troubleshooting Common Issues

- **Application won’t open:** Make sure your Windows system is up to date. Restart your computer and try again.
- **File format not supported:** Verify you are using standard KiCad files (`.sch` for schematics, `.kicad_pcb` for PCBs).
- **Datasheets not downloading:** Check your internet connection and firewall settings.
- **Installation fails:** Run the installer as an administrator by right-clicking the file and selecting “Run as administrator”.
- **Component source finder shows no results:** Check that you entered the correct component name or ID.
